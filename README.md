# PranaGuru Elite PWA

Upload deze bestanden naar de root van je GitHub Pages repo:

- `index.html`
- `manifest.json`
- `sw.js`
- `icons/`
- `assets/`

Open daarna de GitHub Pages URL in Safari op iPhone, tik Delen, kies Zet op beginscherm. De service worker activeert na de eerste online load en maakt de app offline bruikbaar.
