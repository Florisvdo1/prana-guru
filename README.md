## Quick Start (Local)

1. Open `index.html` in Safari/Chrome (best on iPhone simulator or real device)
2. Allow microphone if you want voice guidance (optional)
3. Tap any exercise → choose duration → Start
4. Follow the synced visuals + text + sounds

## Deploy to GitHub (Recommended)

```bash
git init
git add .
git commit -m "PranaGuru v1.0 - Guru level breathing app"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/prana-guru.git
git push -u origin main
```

Then enable **GitHub Pages** (Settings → Pages → Deploy from branch `main` /root).

## Deploy to Vercel / Netlify (One-Click)

- Drag the entire folder to [vercel.com/new](https://vercel.com/new) or [netlify.com/drop](https://netlify.com/drop)
- Done. Instant HTTPS + custom domain support.

## PWA Installation

On iPhone:
1. Open in Safari
2. Tap Share → "Add to Home Screen"
3. Launch from home screen for full immersive experience (no browser UI)
